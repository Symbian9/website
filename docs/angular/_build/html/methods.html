Стандартные методы модуля¶

Вспомогательные методы¶

angular.bind(self, fn, args)¶

карринг для функции fn

angular.bootstrap(element[, modules])¶

используется для старта фреймфорка вручную

angular.callbacks()¶

объект-коллекция колбэков для JSONP

angular.copy(source, destination)¶

полное копирование(deep copy) объекта

angular.element(selector)¶

Селектор из jQuery

angular.equals(o1, o2)¶

сравнение значений/объектов

angular.extend(dst, src[, src2[, src3...]])¶

расширение объекта

angular.forEach(array, callback[, context])¶

Цикл по массиву

Arguments:
  • array (array) – массив
  • callback – обработчик
 
angular.forEach(['1', '2'], function(item){
    console.log(item);
})

// 1
// 2
angular.fromJson()¶

конвертация из JSON

angular.identity(value)¶

создает функцию, которая вернет значение(используется как обертка для мест, где нужно передавать строго функцию)

angular.injector(modules)¶

создает функцию-инжектор, которая может быть использована для получения сервисов

angular.isUndefined, isDefined, isString, isFunction, isObject, isNumber, isElement, isArray, isDate()¶

методы проверки принадлежности типу

angular.lowercase()¶

перевод в нижний регистр

angular.noop()¶

функция “пустышка”, которую можно использовать как заглушку для колбэков

angular.toJson()¶

конвертация в JSON

angular.uppercase()¶

перевод в верхний регистр

angular.version()¶

версия продукта

Методы работы с модулем приложений¶

angular.module(name, requires)¶

Объявляет модуль в приложений и возвращает его экземпляр

Arguments:
  • name (str) – название модуля
  • requires (array) – список зависимостей
 
var app = angular.module('muModule', [])
app.controller(name, callback)¶

Объявляет контроллер в модуле

Arguments:
  • name (str) – название модуля
  • callback – обработчик
 
app.controller(
    'myController',
    function(){
        ...
    }
)
app.constant(key, value)¶

Объявляет константу модуля

Arguments:
  • name (str) – константа
  • val – значение
 
app.constant('CONST', 123)
app.directive(name, callback)¶

Объявляет директиву в модуле

Arguments:
  • name (str) – название
  • callback – обработчик
 
app.directive(
    'myController',
    function(){
        restrict: 'A'
    }
)
