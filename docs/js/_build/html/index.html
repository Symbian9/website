Конспекты JavaScript¶

// подключение в HTML
<script type='text/javascript'>
    alert('Hello World!');
</script>

<script type='text/javascript' src='app.js'></script>
  • Основные объекты
    • Array - массивы
    • Boolean - логический тип данных
    • Date - объект дата
    • Error - исключение
    • EvalError - исключение, генрируемое функцией eval
    • RangeError - генерируется, когда число выходит из допустимого диапазона
    • ReferenceError - генерируется при попытке чтения несуществующей переменной
    • SyntaxError - генерируется при синтаксической ошибке
    • TypeError - генерируется, когда значение имеет неверный тип
    • URIError - генерируется методами кодирования и декодирования
    • Function - функция
    • JSON
    • Infinity - бесконечность
    • Math - математические операции
    • NaN - нечисло
    • Number - число
    • RegExp - Регулярные выражения
    • String - строки
    • Symbol
    • Object - объект
    • Операторы
    • undefined - неопределнное значение
  • DOM - дерево html документа
    • ApplicationCache - прикладной интерфейс управления кэшем приложений
    • ArrayBuffer - последовательность байтов фиксированной длины
    • ArrayBufferView - общие свойства типов, основанных на ArrayBuffer
    • Attr - атрибут элемента документа
    • Audio
    • BeforeUnloadEvent - объект Event для событий выгрузки
    • Blob - блок двоичных данных
    • BlobBuilder - используется для создания новых Blob
    • Button - html элемент button
    • Canvas - канвас страницы
    • Console - консоль браузера
    • Document(HTMLDocument) - текущий документ страницы
    • Element(HTMLElement) - узел дом дерева документа
    • HTMLFormElement
    • HTMLInputElement
    • HTMLOptionElement (Option)
    • HTMLSelectElement
    • HTMLTableCellElement
    • HTMLTableElement
    • HTMLTableRowElement
    • HTMLTableSectionElement
    • HTMLTextAreaElement
    • History - журнал посещений объекта Window
    • Image(HTMLImageElement) - HTML img
    • LocalStorage - key-value хранилище
    • Location - адрес текущей страницы
    • Navigator - свойства браузера
    • Screen - информация о дисплее
    • SessionStorage - key-value хранилище
    • Window - окно, вкладка или фрейм веб-броузера
    • AJAX - Asynchronous Javascript and XML - «асинхронный JavaScript и XML»
    • CSSRule - правило в таблице стилей CSS
    • CSSStyleDeclaration - набор css-атрибутов и их значений
    • CSSStyleSheet - таблица стилей CSS
    • CanvasGradient - цветной градиент для использования в канвасе
    • CanvasPattern - шаблон заполнения холста на основе готового изображения
    • CanvasRenderingContext2D - объект, используемый для создания изображений
    • ClientRect - прямоугольник, описанный вокруг элемента
    • CloseEvent - определяет, был ли закрыт веб-сокет без ошибок
    • Comment - коментарии в доме
    • ConsoleCommandLine - утилита для работы с окном консоли
    • Context - контекст канвас страницы
    • DOMException - исключение, возбуждаемое web api
    • DOMImplementation - глобальные методы DOM
    • DOMSettableTokenList - список лексем с настраиваемым строковым значением
    • DOMTokenList - множество лексем, разделенных пробелами
    • DataTransfer - передача данных в операциях драгндроп мышью
    • DataView - реализует чтение/запись значений в ArrayBuffer
    • DocumentFragment - смежные узлы и их поддеревья
    • DocumentType - doctype документа
    • ErrorEvent - необработанное исключение в фоновом потоке
    • Event - информация о сбоытии
    • EventTarget - comet-соединение с http-сервером
    • EventTarget - объект, способный принимать события
    • FieldSet - HTML fieldset
    • File - файл в локальной файловой системе
    • FileError - ошибка, возникшая во время чтения файла
    • FileReader - асинхронный интерфейс чтения объекта File или Blob
    • FileReaderSync - синхронный интерфейс чтения объекта File или Blob
    • Form - HTML form
    • FormControl - общие особенности всех элементов форм
    • FormData - тело HTTP-запроса multipart/form-data
    • FormValidity - реализует проверку элемента формы
    • Geocoordinates - географическре местонахождение
    • Geolocation - позволяет получить широту и долготу местоположения пользователя
    • GeolocationError - ошибка возникающая в ходе определения местонахождения
    • Geoposition - информация о местонахождении с указанием времени определения
    • HTMLFormControlsCollection - объект, типа массив, содержащий элементы формы
    • HTMLOptionsCollection - коллекция элементов Option
    • HashChangeEvent - объект события, поставляемый по событию hashchange
    • IFrame - HTML iframe
    • ImageData - массив пискелей канваса
    • Input - HTML input
    • MediaElement - элемент проигрывателя
    • MediaError - ошибка в элементе audio, video
    • MessageChannel - пара соединенных объектов MessagePorts
    • MessageEvent - сообщение из другого контекста выполнения
    • MessagePort - передает асинхронные сообщения
    • Meter - графический индикатор, шкала
    • MouseEvent
    • NamedNodeMap
    • Node - узел дом дерева документа
    • NodeList (HTMLCollection) - живой список узлов
    • Option - HTML option
    • Output - HTML output
    • PageTransitionEvent - объект события для событий «pageshow» и «pagehide»
    • PopStateEvent - событие перемещения по истории посещений
    • ProcessingInstruction - инструкция обработки в XML-документе
    • Progress - индикатор хода выполнения операции
    • ProgressEvent - событие продолжения загрузки, выгрузки или чтения файла
    • Script - HTML-script
    • Select - графический список для выбора
    • Storage - хранилище пар имя/значение на стороне клиента
    • StorageEvent - хранилище пар имя/значение на стороне клиента
    • Style - HTML style
    • Table - HTML table
    • TableCell - ячейка HTML таблицы
    • TableRow - HTML tr
    • TableSection - раздел верхнего или нижнего колонтитула либо тела таблицы
    • Text - текстовая последовательность в документе
    • TextArea - многострочная область ввода текста
    • TextMetrics - определяет размеры текстовой строки
    • TimeRanges - множество фрагментов мультимедийных данных
    • TypedArray - массивы с двоичными элементами фиксированного размера
    • URL - методы для работы с URL-адресами, ссылающимися на объекты Blob
    • Video - HTML video
    • WebSocket - двунаправленное сетевое соединение, подобное сокету
    • Worker - фоновый поток выполнения
    • WorkerGlobalScope
    • WorkerLocation - URL-адрес главного сценария в фоновом потоке выполнения
    • WorkerNavigator - информация о броузере для фонового потока выполнения
    • XMLHttpRequest(AJAX) - позволяет выполнять HTTP-запросы и получать ответы
    • XMLHttpRequestUpload
