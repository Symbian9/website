readyState¶

Статус обработки запрос

  • 0 - open еще не вызывался
  • 1 - open вызван, send ещё не вызван
  • 2 - send вызван, но ответа пока нет
  • 3 - идет прием данных
  • 4 - ответ от сервера получен
