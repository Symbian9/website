NodeList (HTMLCollection) - живой список узлов
==============================================

Живой - т.е. объект не статический, а динамический. Изменяется при каждом изменении дом дерева


.. js:class:: NodeList()
.. js:class:: HTMLCollection()


    .. js:attribute:: length
        
        Ко­ли­че­ст­во эле­мен­тов в кол­лек­ции.


    .. js:function:: item(unsigned long index)
        
        Воз­вра­ща­ет эле­мент кол­лек­ции :js:class:`Element`, рас­по­ло­жен­ный в по­зи­ции index, или null, ес­ли ин­декс index вы­хо­дит за гра­ни­цы мас­си­ва. Этот ме­тод мож­но не вы­зы­вать яв­но, а ука­зать ин­декс в квад­рат­ных скоб­ках (как для мас­си­ва).


    .. js:function:: namedItem(string name)
        
        Воз­вра­ща­ет пер­вый эле­мент :js:class:`Element` из кол­лек­ции, имею­щий зна­че­ние name в ат­ри­бу­те id или name ли­бо null, ес­ли та­кой эле­мент от­сут­с
