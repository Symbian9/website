Date - объект дата
==================

.. js:class:: Date()

    Дата

    .. code-block:: js

        var g = new Date();
        var g = new Date(year, month, date, hours, minutes, seconds, ms);
        var g = new Date(2013, 1, 1);
        var g = new Date(5000);
        var g = new Date('2013-01-01');
        var g = new Date('December 25, 2013 21:46:10');
        var g = new Date('03/09/2000');
