Math
====

Основные методы
---------------

.. js:function:: Math.floor(var)

    Округление в большую сторону

.. js:function:: Math.ceil(var)

    Округление в меньшую сторону

.. js:function:: Math.max(var1, var2, ....)

    Возвращает максимум из переданных аргументов

.. js:function:: Math.min(var1, var2, ....)

    Возвращает минимум из переданных аргументов

.. js:function:: Math.pow(var, pow)

    Возведение в степень

.. js:function:: Math.random()

    Возвращает случайное число из промежутка от 0 до 1

.. js:function:: Math.round(var)

    Нормальное округление
