String - строки¶

class String()¶

Строки

Наследник Object()

var a = new String('test'),
    string = 'ilnurgi',
    s += 's';

// в переменную а мы можем добавить свои атрибуты и методы

'\u1552'.length
//1

'il\'s'.length
//3

'ilnurgi' + 123;
// 'ilnurgi123'

'12'/2 + 1;
// 7

'day' * 2;
// NaN

'a' < 'b';
// true

'c' < 'b';
// false
length()¶
'ilnurgi'.length;
// 7
charAt(index)¶

Символ из строки

'ilnurgi'.charAt(2);
// 'n'

'ilnurgi'.charAt(200);
// ''

'ilnurgi'.charAt(-1);
// ''
charCodeAt(index)¶

Код символа из строки

'!'.charCodeAt(0);
// 33
concat(index)¶

Конкатенация одного или нескольких значений со строкой

fromCharCode(code)¶

Статический метод, возвращает новую строку

String.fromCharCode(33); // '!'
String.fromCharCode(104, 101, 108, 108, 111); // "hello"
indexOf(substr[, start_pos])¶

Возвращает индекс вхождения подстроки в строку.

'строка'.indexOf("ока");
// 3
lastIndexOf(substr[, start_pos])¶

Возвращает индекс вхождения подстроки в строку с конца.

'строка'.lastIndexOf("ока");
// 3
localeCompare(str)¶

Сравнивает строки с учетом порядка следования символов национальных алфавитов

match(str)¶

Поиск совпадений в строке, возвращает массив

replace(str1, str2)¶

Поиск и замена

search(str)¶

Поиск совпадений в строке, возвращает индекс

slice(start[, end])¶

Поиск совпадений в строке, возвращает индекс

split(разделитель, лимит)¶

Разбиение строки на массив

substr(start[, length])¶

Срез строки

substring(start[, length])¶

Срез строки

toLowerCase()¶

Преобразует строку в нижний регистр

toUpperCase()¶

Преобразует строку в верхний регистр

trim()¶

Возвращает копию строки, с удаленными пробелами вначале и в конце
