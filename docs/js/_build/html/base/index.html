Основные объекты¶

  • Array - массивы
  • Boolean - логический тип данных
  • Date - объект дата
  • Error - исключение
  • EvalError - исключение, генрируемое функцией eval
  • RangeError - генерируется, когда число выходит из допустимого диапазона
  • ReferenceError - генерируется при попытке чтения несуществующей переменной
  • SyntaxError - генерируется при синтаксической ошибке
  • TypeError - генерируется, когда значение имеет неверный тип
  • URIError - генерируется методами кодирования и декодирования
  • Function - функция
  • JSON
  • Infinity - бесконечность
  • Math - математические операции
    • Тригонометрические функции
  • NaN - нечисло
  • Number - число
  • RegExp - Регулярные выражения
  • String - строки
  • Symbol
  • Object - объект
    • Методы, доступные только в объекте Object
    • Дескрипторы свойств
  • Операторы
    • Логические
    • Битовые
    • if
    • for
    • while
    • switch
    • break
    • continue
    • instanceof
    • try, catche, finally
    • throw
    • in
    • ?
    • do - while
    • метки
    • delete
  • undefined - неопределнное значение
