String - строки¶

class String()¶

Строки

var a = new String('test');
var string = 'ilnurgi';

// в переменную а мы можем добавить свои атрибуты и методы

'\u1552'.length
//1

'il\'s'.length
//3

'ilnurgi' + 123;
// 'ilnurgi123'

'12'/2 + 1;
// 7

'day' * 2;
// NaN

'a' < 'b';
// true

'c' < 'b';
// false
length()¶
'ilnurgi'.length;
// 7
charAt(index)¶

Символ из строки

'ilnurgi'.charAt(2);
// 'n'

'ilnurgi'.charAt(200);
// ''

'ilnurgi'.charAt(-1);
// ''
charCodeAt(index)¶

Код символа из строки

'!'.charCodeAt(0);
// 33
fromCharCode(code)¶

Символ по коду

String.fromCharCode(33);
// '!'
indexOf(substr[, start_pos])¶

Возвращает индекс вхождения подстроки в строку.

'строка'.indexOf("ока");
// 3
lastIndexOf(substr[, start_pos])¶

Возвращает индекс вхождения подстроки в строку с конца.

'строка'.lastIndexOf("ока");
// 3

Поиск и замена

Arguments:
