//xen, xan, xon, xyn

/[^b]log/
//dlog, flog

/[A-Z]/

экранирование
. любой символ
0 и более
1 и более
? 0 или 1

exec()¶

Поиск совпадений в строке, возвращает массив (функция генератор)

var a = /\d/g;

a.exec('kj5k3');
//['5', index: 2, input:]

a.test('123');
//true
test(string)¶

Есть ли совпадение, возвращает true|false

var a = /\d/;

a.test('qw');
//false

a.test('123');
//true
