Теги форматирование шрифтов
---------------------------

=================== =====
Тег                 Описание
=================== =====
acronym             акроним(сокращение)
b                   полужирный
big                 увеличен относительно стандартного размера
blink               разрыв строки
cite                цитата
code                исходный текст программы
dfn                 определение термина (IE)
em                  курсивый
i                   курсивый
kbd                 пример ввода пользователя
samp                пример исходного кода программы
small               уменьшен относительно стандартного размера
strike              зачеркнутый
strong              выделенниый текст
sub                 верхний индекс
sup                 нижний индекс
tt                  моноширный шрифт (все символы одинаковой ширины)
u                   подчеркнутый
var                 имя переменной
=================== =====
