details
=======

Область, которая может разворачиваться и сворачиваться.

<summary> — элемент управления <details> (для текста "подробнее" и тд).


open
----

Развернуть область

.. code-block:: html

    <details>
        <summary>Дополнительная информация</summary>
        <p>....</p>
    </details>
