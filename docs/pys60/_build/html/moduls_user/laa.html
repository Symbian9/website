laa¶

Модуль для запуска любых приложений включая java приложения авторы: Maxim6630, kolayuk, watt

laa.execute(uid)¶
Parameters:uid (int) – uid приложения

Запускает приложение

>>> laa.execute(0xa0000bcd)
# запускает x-plore
laa.execute_param(uid, param)¶
Parameters:
  • uid (int) – uid приложения
  • param (str) – параметры запуска

Запускает приложение c параметром

>>> laa.execute_param(0x10008d39, u'http://www.ilnurgi.ru/')
# запускает стандартный браузер и переходит по ссылке
laa.about()¶

Возвращает информацию о модуле

>>> laa.about()
# 'LAA - launch any app by Maxim6630 , v1.02'
