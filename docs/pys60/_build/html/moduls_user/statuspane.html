statuspane.set(value)¶

  • 0x8cc0168 - R_AVKON_STATUS_PANE_LAYOUT_SMALL_WITH_SIGNAL_PANE
  • 0x8cc0169 - R_AVKON_STATUS_PANE_LAYOUT_SMALL_WITH_SIGNAL_PANE_MIRRORED
  • 0x8cc016a - R_AVKON_STATUS_PANE_LAYOUT_USUAL_MIRRORED
  • 0x8cc016b - R_AVKON_STATUS_PANE_LAYOUT_IDLE_MIRRORED
  • 0x8cc016c - R_AVKON_STATUS_PANE_LAYOUT_POWER_OFF_RECHARGE_MIRRORED
  • 0x8cc0170 - R_AVKON_STATUS_PANE_LAYOUT_VT
  • 0x8cc0171 - R_AVKON_STATUS_PANE_LAYOUT_VT_MIRRORED
  • 0x8cc0179 - R_AVKON_STATUS_PANE_LAYOUT_USUAL_WITH_BATTERY_PANE
  • 0x8cc018d - R_AVKON_STATUS_PANE_LAYOUT_USUAL_FLAT
  • 0x8cc018e - R_AVKON_STATUS_PANE_LAYOUT_IDLE_FLAT
  • 0x8cc0032 - R_AVKON_STATUS_PANE_LAYOUT_IDLE
  • 0x8cc0033 - R_AVKON_STATUS_PANE_LAYOUT_USUAL
