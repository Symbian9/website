keypress¶

keypress.simulate_key(scan, scan)¶
Parameters:scan – сканкод кнопки

Симулирует нажатие клавиши.

Note

После нажатия необходимо вставить задержку ao_sleep(0.1) иначе не работает второе нажатие.

keypress.simulate_key_mod(scan, scan, mode)¶
Parameters:
  • scan – сканкод кнопки
  • mode – 0|1 нажать|снять удержание

Симулирует удержание клавиши

keypress.simulate_keyMenu()¶
