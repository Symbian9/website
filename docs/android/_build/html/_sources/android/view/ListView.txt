.. _android_view_ListView:

android.view.ListView
=====================

.. py:class:: android.view.ListView

    Наследник :ref:`android.view.View`. Группа представлений, которая формирует вертикальный список элементов, отображая их в виде строк внутри списка. Простейший объект ListView использует TextView для вывода на экран значений toString, принадлежащих элементам массива.
