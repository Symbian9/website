android.widget.TableLayout
==========================

.. py:class:: android.widget.TableLayout

    Позволяет размещать Представления с помощью сетки, состоящей из строк и столбцов. При этом столбцы могут либо автоматически растягиваться, либо оставаться постоянной ширины.
