android.widget.Spinner¶

class android.widget.Spinner¶

Наследник android.view.View. Составной элемент, отображающий TextView в сочетании с соответствующим Представлением ListView, которое позволяет выбрать элемент списка для отображения в текстовой строке. Сама строка состоит из объекта TextView, показывающего текущий выбор, и кнопки, при нажатии которой всплывает диалог выбора.
