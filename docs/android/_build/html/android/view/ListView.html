android.view.ListView¶

class android.view.ListView¶

Наследник android.view.View. Группа представлений, которая формирует вертикальный список элементов, отображая их в виде строк внутри списка. Простейший объект ListView использует TextView для вывода на экран значений toString, принадлежащих элементам массива.
