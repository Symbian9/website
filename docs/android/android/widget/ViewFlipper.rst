android.widget.ViewFlipper
==========================

.. py:class:: android.widget.ViewFlipper

    Наследник :ref:`android.view.View`. Группа представлений, позволяющая определить набор элементов и горизонтальную строку, в которой одновременно может выводиться только одно Представление. При этом переходы между отображаемыми элементами осуществляются с помощью анимации.
