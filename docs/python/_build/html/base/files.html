file - файлы¶

Функции для работы с файлами¶

open(path [, mode='r' [, buffering=-1 [, encoding=None [, errors=None [, newline=None [, closefd=True]]]]]]))¶
Параметры:
  • path (str) – путь к открываемому файлу
  • mode (str) – режим работы, r,w,a, (t, b)
  • buffering (int) – определяет размер буферизации
  • encoding (str) – кодировка файла
  • errors (str) – политику обработки ошибок кодирования символов и может иметь одно из следующих значений: ‘strict’, ‘ignore’, ‘replace’, ‘backslashreplace’ или ‘xmlcharrefreplace’.
  • newline (str) – определяет поведение режима поддержки универсального символа перевода строки и может иметь значение None, ‘’, ‘n’, ‘r’ или ‘rn’.
  • closefd (bool) –

    логический флаг, который определяет, должен ли фактически закрываться дескриптор файла при вызове метода close().

    • w - запись, содержимое исчезает
    • r - чтение
    • a - добавление/запись файла, содержимое не изменяется
    • r+ - чтение/запись, содержимое не меняется
    • w+ - запись, содержимое исчезает
    • r+b - чтение/запись двоичного содержимого, содержимое сохраняется
    • w+b - чтение/запись двоичного содержимого, содержимое удаляется
    • rb - чтение двоичного файла
    • wr - запись двоичного файла, содержимое исчезает

Возвращает файловый объект.

>>> f = open('ilnurgi.txt')
>>> type(f)
type 'file'

Методы файловых объектов¶

file.close()¶

Закрывает файловый объект

>>> f.close()
file.fileno()¶

Возвращает целочисленный дескриптор файла

file.flush()¶

Выталкивает буферы вывода

file.isatty()¶

Интерактивный терминал

file.read([n])¶
Параметры:n (int) – количество читаемых байтов

Возвращает строку, прочитанные данные из файла.

>>> f.read()
'123'
file.readline([n])¶
Параметры:n (int) – количество читаемых байтов

Возвращает строку, прочитанную из файла построчно.

file.readlines([n])¶

Возвращает список, строки из файла

Параметры:n (int) – количество читаемых байтов 
>>> f.readlines()
['1', '2']
file.seek(offset[, whence])¶
Параметры:
  • offset (int) – позиция чтения
  • whence (int) –

    относительно чего отсчитывается начальная позиция

    • io.SEEK_SET или 0 - новое положение отсчитывается от начала файла;
    • io.SEEK_CUR или 1 - от конца файла;
    • io.SEEK_END или 2 - от текущего положения в файле;

Перемещает курсор в файле на определенную позицию

file.tell()¶

Возвращает текущее положение в файле в байтах от начала файла.

file.truncate(size)¶
Параметры:size (int) – усекаемый размер

Усекает файл до заданного размера

file.write(str)¶

Записывает в файл строку

>>> f.write('string')
file.writelines(iter)¶

Записывает в файл список строк

>>> f.writelines('qwert')
file.closed¶

Состояние файла, закрыт, открыт

file.mode¶

Режим ввода-вывода

file.softspace¶

Логическое значение, сообщающее, должна ли инструкция print выводить пробел перед очередным значением. Классы, имитирующие файлы, должны предоставлять атрибут с этим именем, доступный для записи, инициализируемый нулем (только в Python 2).

file.newlines¶

Когда файл открывается в режиме поддержки универсального символа перевода строки, этот атрибут будет содержать представление символа окончания строки, фактически используемого в файле. Значением атрибута может быть None, которое говорит о том, что никаких символов завершения строки не было встречено; строка, содержащая ‘n’, ‘r’ или ‘rn’, или кортеж со всеми встреченными символами завершения строки.

file.encoding¶

Строка с названием кодировки файла, если определена (например, ‘latin-1’ или ‘utf-8’). Если кодировка не используется, этот атрибут имеет значение None.

file.name¶

название файла

file.buffer¶

буфер файлового объекта

>>> f = open(r"file.txt", "w", encoding="cp1251")
>>> f.buffer.write(bytes("Cтpoкa", "ср1251"))
6
>>> f.close()
