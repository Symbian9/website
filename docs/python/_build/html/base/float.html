float - вещественное число¶

>>> 10., 11E20, 2.5e-12
10., 11E20, 2.5e-12

>>> 0.3 - 0.1 - 0.1 - 0.1
-2.7755575615628914е-17
class float

Вещественное число. Для точных расчетов необходимо использователь модуль decimal или модуль fractions для рациональных чисел.

>>> type(5.1), type(8.5e-3)
(<class 'float'>, <class 'float'>)

Функции для работы с float¶

float([obj])
Параметры:obj (str , int) – аргумент

возвращает вещественное число, преобразованое из аргумента

>>> float(7), float("7.1")
(7 .о, 7 .1)
>>> float("Infinity"), float("-inf")
(inf, -inf)
>>> float ("Infinity") + float ("-inf")
nan
