create_arc(x1, y1, x2, y2, **kwargs)¶

Рисует круг, вписанную в прямоугольник, и возвращает id

Параметры:
  • start – угол, начало рисования
  • extent – угол, величина рисуемой окружности
  • fill (str) – цвет заполнения
  • outline (str) – цвет контура
  • width (int) – ширина контура
  • tag (str) – название объекта
