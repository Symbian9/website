Text¶

class Text(widget, **kwargs)¶

Виджет, многострочное текстовое поле. Поддерживает те же методы что и Entry, но индексы передаются в виде ‘1.0’

Параметры:
  • widget – родительский виджет
  • height (int) – высота виджета
  • width (int) – ширина виджета

tag_add(name, start, end)¶

Добавляет тег в текст

Параметры:
  • name (str) – название тега
  • start – начало
  • end – конец
 
>>> t.tag_add('tag', tk.SEL_FIRST, tk.SEL_LAST)

Конфигурирует тег

Параметры:

  • background (str) – фон
  • foregraund (str) – текст
  • font (typle) – параметры шрифта
 
>>> t.tag_config('tag', background='black', foregraund='red', font=('times', 14, 'italic'))
