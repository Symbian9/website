Progressbar¶

class Progressbar(widget, **kwargs)¶

Виджет, прогрессбар

Параметры:
  • widget – родительский виджет
  • height (int) – длина
  • maximum (int) – максимум пунктов в виджете
start([time])¶

Запускает прогрессбар

Параметры:time (int) – интервал между загрузками, мс
step()¶

Делает шаг в прогрессбаре

stop()¶

Останавливает загрузку
