Treeview¶

class Treeview(widget, **kargs)¶

Виджет, дерево

Параметры:
  • widget – родительский виджет
  • columns (tuple) – список колонок
  • selectmode – tk.EXTENDED - множественный выбор, tk.BROWSE - выбор 1 строки, tk.NONE - без выбора вообще
column(tag, **kwargs)¶

Добавляет в таблицу колонку

Параметры:
  • tag – тег колонки
  • anchor (str) – выравнивание содержимого ячеек
  • minwidth (int) – минимальная ширина колонки
  • width (int) – ширина колонки
heading(tag, **kwargs)¶

Устанавливает заголовок столбца

Параметры:
  • tag – тег колонки
  • text (str) – заголовок колонки
  • img (PhotoImage) – вставляет картинку в заголовок
insert(tag, index, **kwargs)¶

Добавляет в таблицу строку

Параметры:
  • tag – тег строки, в которую добавляется новая строка
  • index – индекс позиции, куда вставляется строка (tk.END)
  • text (str) – заголовок строки
  • values (tuple) – кортеж строк, значений
  • img (PhotoImage) – вставляет картинку в строку
