Scale¶

class Scale(widget, **kwargs)¶

Виджет, позволяет устанавливать некоторое значение

Параметры:
  • widget – родительский виджет
  • from – начальная позиция
  • to – конечная позиция
  • resolution – шаг
  • label – подпись виджета
  • tickinterval (int) – градуировка бегунка
  • sliderlength (int) – длина бегунка
  • orient – ориентация бегунка tk.HORIZONTAL
  • length – длина бегунка
coords()¶

Возвращает кортеж (х, у) положение слайдера относительно шкалы.
