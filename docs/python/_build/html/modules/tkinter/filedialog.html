filedialog¶

filedialog.askdirectory()¶

Открывает окно выбора папки, возвращает строку, путь к папке

filedialog.askopenfilename()¶

Открывает окно выбора файла, возвращает строку, путь к файлу

filedialog.askopenfilenames()¶

Открывает окно выбора файла, возвращает список строк, путей к выбранным файлам

filedialog.askopenfile()¶

Открывает окно выбора файла, возвращает файловый объект

filedialog.asksaveasfile()¶

Открывает окно сохранения файла, возвращает путь к файлу
