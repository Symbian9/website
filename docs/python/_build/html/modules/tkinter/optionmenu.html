OptionMenu¶

class OptionMenu(widget, var, *list)¶

Меню, список для выбора

Параметры:
  • widget – виджет для встраивания
  • var – переменная для установки
  • list – список для выбора
 
s = tk.StringVar()
s.set(u'Красный')
lis = [u'Красный', u'Зеленый']
sp = tk.OptionMenu(window, s, *lis)
