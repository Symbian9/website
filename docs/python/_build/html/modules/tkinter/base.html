Основные методы виджетов¶

bell()¶

Издает звук

bind(event, callback)¶

Привязывает событие виджета к обработчику. Передает в обработчик в качестве аргумента. объект :py:class::tkinter.Event

>>> widget.bind('<Button-1>', callback)
  • Button-1 - нажатие левой кнопки мыши
  • Button-2 - нажатие средней кнопки мыши
  • Button-3 - нажатие правой кнопки мыши
  • Double-Button-1 - двойное нажатие
  • ButtonRelease-1 - отпускание кнопки
  • B1-Motion - движение мыши с зажатой кнопкой
  • Enter - наведение курсора на тег
  • Leave - уход курсора от тега
  • r - кнопка на клавиатуре
  • Shift_L - левый шифт
  • Alt_L
  • Control_R
  • Return
  • Caps_Lock
  • Escape
  • Prior - PageUp
  • Next- PageDown.
  • Up
  • Down
  • Left
  • Right
destroy()¶

Уничтожает виджет

>>> window.destroy()
config(**kwargs)¶

Конфигурирует виджет

Параметры:
  • bg (str) – цвет фона
  • command (func) – обработчик
  • cursor (str) – тип курсора при наведении (pencil, cross, watch, hand2)
  • fg (str) – цвет текста
  • font (str) – шрифт, ‘Helvetica 20 italic’, (bold, italic, underline, overstrike, roman, normal)
  • justify – выравнивание текста (tk.LEFT, tk.RIGHT, tk.CENTER)
  • height (int) – высота
  • width (int) – ширина
  • relief – стиль отображения виджета, tk.SOLID, tk.SUNKEN, tk.FLAT, tk.RIDGE, tk.RAISED, tk.GROOVE.
  • state – tk.NORMAL, tk.ACTIVE, tk.DISABLED, ‘readonly’
  • yscrollcommand (func) – обработчик
  • xscrollcommand (func) – обработчик
focus()¶

Устанавливает фокус на виджете

get()¶

Возвращает значение виджета

protocol(event, func)¶

Вещает обработчик события виджета

Параметры:
  • event – ивент, (‘WM_DELETE_WINDOW’)
  • func – обработчик
set()¶

Устанавливает значение виджета

tag_bind(tag, event, callback)¶

Привязывает обработчик тега с ивентом

Основные методы расположения виджетов¶

widget.pack(**kwargs)¶
Параметры:
  • side – расположение, tk.TOP, tk.BOTTOM, tk.LEFT, tk.RIGHT
  • fill – растягивание, tk.X, tk.Y, tk.BOTH
  • expand – устанавливает параметры растягивания при изменении окна, tk.YES
  • anchor – расположение виджета в виджете, tk.N, rk.S, tk.W, tk.E
  • padx – внутренний отступ виджета
  • pady – внутренний отступ виджета
widget.grid(**kwargs)¶
Параметры:
  • row – номер строки
  • column – номер колонки
