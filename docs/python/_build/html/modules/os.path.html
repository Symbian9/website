Модуль встроен в модуль os и позволяет работать с путями к файлам и папкам

basename(path) Возвращает основное имя пути path. Это второй элемент пары split(path) os.path.basename (‘c:\system\apps\Python\Python.app’) ‘Python.app’ dirname(path) Возвращает имя каталога пути path. Это первый элемент пары split(path) os.path.dirname (‘c:\system\apps\Python\Python.app’) ‘c:\system\apps\Python’ exists(path) Возвращает 1, если файл или каталог с именем path существует

getatime(path) Возвращает время последнего доступа к файлу или папке с именем path

getmtime(path) Возвращает время последнего внесения изменения в файл или папку с именем path

getsize(path) Возвращает размер файла или папки с именем path

join(path1, path3,...) Объединяет пути path. os.path.join(‘c:\’, ‘system\apps\Python\’, ‘Python.app’) ‘c:\system\apps\Python\Python.app’ isfile(path) Возвращает 1, если path указывает на файл

isdir(path) Возвращает 1, если path указывает на папку

split(path) Возвращает кортеж из пары строк - (head, tail). tail - последний компонент пути (имя папки или файла) head - остальная часть пути os.path.split (‘c:\system\apps\Python\Python.app’) (‘c:\system\apps\Python\’, ‘Python.app’) splitdrive(path) Возвращает кортеж из пары строк - (drive, tail). drive - имя диска head - остальная часть пути os.path.splitdrive (‘c:\system\apps\Python\Python.app’) (‘c:\’, ‘system\apps\Python\Python.app’) splitext(path) Возвращает кортеж из пары строк - (root, ext) ext - расширение файла head - остальная часть пути os.path.splitext (‘c:\system\apps\Python\Python.app’) (‘c:\system\apps\Python\Python’, ‘.app’) walk(path, visit, arg) Вызывает функцию visit передавая ей параметры def listfiles(arg ,dirname , fnames) :
