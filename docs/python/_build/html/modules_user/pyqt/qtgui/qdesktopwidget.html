QDesktopWidget - объект, рабочий стол операционной системы¶

class QtGui.QDesktopWidget¶

Данный объект можно получить методом QtGui.QApplication.desktop()

Наследник QtGui.QWidget

availableGeometry()¶

Возвращает QtCore.QRect, содержащий координаыт только доступной части экрана

height()¶

Возвращает число, высоту всего экрана

screenGeometry()¶

Возвращает QtCore.QRect, содержащий координаыт всего экрана

width()¶

Возвращает число, ширину всего экрана
